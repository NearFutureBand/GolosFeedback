Usage example injection of the GolosFeedback

Put this <script src="https://golosfeedback.com/js/inject.js"></script> tag to the <head> of your page

Add this button on your page

<button type="button" data-toggle="modal" data-target=".bd-example-modal-lg"></div>

Set up a standard bootstrap modal window

<div class="modal fade modal-lg-golos-feedback" tabindex="-1" role="dialog" aria-labelledby="GolosFeedback" aria-hidden="true">
    <div class="modal-dialog modal-lg">
        <div class="modal-content golos-feedback-container">
  
        </div>
    </div>
</div>

That`s all folks!
