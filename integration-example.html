Usage example GolosPolls-Injection

Put this <script src="https://golosfeedback.com/js/inject.js"></script> tag to the <head> of your page

Put this div and script tags

<div class="gFeedback"></div>

<script type="text/javascript"> var gFeedbackWidth = '1000';</script>

to the place, where the Poll block will be

That`s all folks!
